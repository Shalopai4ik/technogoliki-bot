# 🤖 Technogoliki Telegram Bot

Бот для управления каналом Technogoliki с системой рефералов и предложки постов.

## ✨ Функционал
- 📝 Система предложки постов по темам
- 🔗 Реферальная система
- 👮 Модерация и админ-панель
- 📊 Мониторинг и статистика
- 📨 Рассылка сообщений
- 🔌 Интеграция с VK (опционально)
